Lab2 - Complete project (implements Lab2.pdf requirements)

How to run:
1. Ensure JDK 17 and Lombok installed in your IDE (enable annotation processing).
2. Import as Existing Maven Project in Spring Tool Suite (STS) or any IDE.
3. Run Lab2Application.java as Spring Boot App or run: mvn spring-boot:run

Endpoints:
- /ctrl/ok   -> Bài 1 (three buttons mapped to m1,m2,m3)
- /param/form and POST /param/save/2021 -> Bài 2
- /product/form and POST /product/save -> Bài 3 & 4
- /product/list?page=1&size=5 -> Bài 6 (20 sample products)
- /a, /b, /c, /d -> Bài 5 (model, redirect attribute, response body)
